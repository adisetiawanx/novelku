# Novelku

## Tech Stack

- Nuxt 3
- Bun

## Features

- OAuth login/register (Google)
- Traditional login/register (email binding)
- Content Management System
- Bulk upload novel

## Installation

prerequired:

- Postgresql and create novelku db
- Google OAuth client
- Setting .env file

```bash
bun install
```

```bash
bun prisma:migrate
```

```bash
bun prisma:generate
```
